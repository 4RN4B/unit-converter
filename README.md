# Gating started with this app

First of all install the node modules after cloning in your system

```
npm install
```

The for run the app in your local system

```
npm start
```

After adding new files run

```
git add .
```

After changing the existing files run

```
git commit -am "Commit name"
```

Push in your github account

```
git push origin master
```

To pull the changes from your GH account

```
git pull origin master
```

After finishing the task you can notify me by creating PR
